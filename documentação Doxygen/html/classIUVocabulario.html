Descrição Detalhada

Interface de Vocabulario de Usuario na camada de Apresentacao.

Contem metodos abstratos para dar inicio a vocabulario de usuario/contas. Tais metodos sao implementados pelo Modulo responsavel e basicamente fazem chegagem de Formato dos dados fornecidos.
