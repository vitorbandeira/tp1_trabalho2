Descrição Detalhada

Interface de Gestao de Usuario na camada de Apresentacao.

Contem metodos abstratos para dar inicio a gestao de usuario/contas. Tais metodos sao implementados pelo Modulo responsavel e basicamente fazem chegagem de Formato dos dados fornecidos.
